Перед запуском бота:
0. Установить python 3.10 и библиотеки из файла requirements.txt
1. Изменить токен, айди и, при необходимости, опции перевода в файле data/config.py
2. Установить mysql и ввести данные по ней в файл data/config.py (инструкция по установке 5 мин. https://www.youtube.com/watch?v=WMGRfexUz64)
3. Запустить файл utils/manage_db.py для создания таблицы
4. Запустить бота двойным щелчком по файлу app.py
